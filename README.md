# Voice Transport Enquiry System 🎙️🚌

A modern, fast, voice-enabled transport enquiry system built with React, Vite, and Tailwind CSS.

## Features
- **Voice Search**: Real-time integration with the browser's native Web Speech API to recognize voice commands and navigate to results.
- **Micro-interactions**: Ripple rings, hover effects, stateful search bars, and pulsing typography to make the UI feel engaging and responsive.
- **SaaS Dashboard Interface**: Ultra-clean, modern, dark-themed UI built over responsive Tailwind components.

## Tech Stack
- **Frontend**: React 18, React Router DOM
- **Build Tool**: Vite
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **API**: Browser Native Web Speech API

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Design Highlights
Inspired by leading minimalist AI tools, featuring pill-shaped search bars, smooth dark mode gradients, rotating query headers, and live-transcription indicators.
